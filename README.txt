MARKETPILOT AI WEBSITE TEMPLATE

What this is:
- A smooth stock-market website template
- Built with plain HTML, CSS, and JS
- Safe for GitHub + Vercel deployment
- No framework, no npm install, no build command needed

FILES:
- index.html
- styles.css
- script.js
- vercel.json

HOW TO DEPLOY TO VERCEL:
1. Unzip this folder.
2. Upload all files into a new GitHub repository.
3. Go to Vercel.
4. Import the GitHub repo.
5. Framework preset: Other
6. Build command: leave blank
7. Output directory: leave blank
8. Deploy

WHY THIS SHOULD DEPLOY CLEAN:
- No React
- No Vite
- No package.json
- No dependency errors
- Static files only

WHAT TO CHANGE FIRST:
- In index.html, replace 'MarketPilot AI' with your brand name if you want.
- Replace the mailto link near the bottom with your real contact link.
- Replace the demo stock content later when live APIs are added.
